### Text Based Protocol vs Binary Based Protocol:

- Text Based Protocol: data is in human-readable text it holds key:value in text like http protocol
    ```http
    GET /products?id=5 HTTP/1.1
    Host: example.com
    User-Agent: Chrome
    ```

- Binary Protocol: sends data as raw bytes, you idetify data by its position, for example first 2 bytes represents source port, second 2 bytes represents destination port,etc...
    ```text
    0x12 0xA7 0x01 0x00 0x05 ...
    ```
